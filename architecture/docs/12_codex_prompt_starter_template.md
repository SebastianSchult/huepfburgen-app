# Codex Prompt – SaaS Starter Template

Create a multi-tenant SaaS starter template.

Frontend:
- React
- TypeScript
- Vite
- Tailwind

Backend:
- Node.js
- Fastify
- Prisma
- PostgreSQL

Architecture:
- multi-tenant
- tenant_id isolation
- JWT auth
- worker with BullMQ

Important:
Use React instead of Angular.
